## Getting Started

First, install dependencies:

```bash
npm install
```

Second, run the development server:

```bash
npm run dev
```

### Seed DB

```bash
npm run seed
```

This will seed the DB with 4 items for the Todo-list

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
